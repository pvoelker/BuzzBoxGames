Buzz Box Games

For use with Brian's Quiz Boxes...

  • News
  • Downloads
  • Feedback

A collection of simple games to play using the USB connection of a Brian's Quiz Box. It's a way to get new quizzers comfortable with using buzzers. It also provides a fun break for even seasoned quizzers.

These are the games:

  • Simon Says - Repeat the paddle press pattern! Each round another paddle press is added. See how long of a pattern you can remember...
  • Reaction Time - See who can buzz in the fastest! Reaction times are given in milliseconds. Winner gets a star...
  • Drop-Tac-Toe - A variation on Tic-Tac-Toe! See if your team (red or green) can get three in a row...

See the Downloads section on links to get the application.
